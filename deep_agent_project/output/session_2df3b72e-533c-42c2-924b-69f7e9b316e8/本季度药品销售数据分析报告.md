# 本季度药品销售数据分析报告

## 一、报告说明

本报告基于企业内部数据库中的药品销售数据，对“本季度”药品销售表现进行汇总分析，重点覆盖总体销售规模、月度趋势、品类结构、单品贡献、区域与渠道分布等维度，以便为后续经营决策、资源投放、品类优化和销售策略制定提供参考。

### 1.1 数据口径说明
- **统计窗口**：2026-01-01 至 2026-03-31
- **实际数据覆盖**：2026-01-05 至 2026-03-15
- **分析对象**：企业药品销售事实数据与药品主数据
- **销售额口径**：销售表中的 `amount` 汇总
- **销售数量口径**：销售表中的 `quantity` 汇总
- **毛利口径**：销售金额 - 销售数量 × 药品单价
- **毛利率口径**：毛利 / 销售金额

需要说明的是，当前数据库仅覆盖季度内的部分时间，且缺少上一季度和去年同期数据，因此环比与同比无法严格计算。本报告以当前可用数据窗口作为“本季度”分析基础。

### 1.2 使用数据表
#### 1）drug_sales（销售事实表）
字段包括：
- `sale_id`：销售单号
- `drug_id`：药品ID
- `sale_date`：销售日期
- `region`：区域
- `customer_type`：客户类型/渠道
- `quantity`：销售数量
- `amount`：销售金额

#### 2）drug_info（药品主数据表）
字段包括：
- `drug_id`：药品ID
- `drug_name`：药品名称
- `generic_name`：通用名
- `category`：品类
- `manufacturer`：生产厂家
- `specification`：规格
- `unit_price`：单价
- `approval_no`：批准文号
- `is_prescription`：是否处方药

#### 3）drug_inventory（库存表）
本次未直接用于核心指标，但可在后续扩展供销存分析时使用。

- 对**呼吸系统、降糖药**等占比偏低品类，考虑通过组合销售或重点市场推广提升规模。

### 7.1 毛利分析结果

- **毛利：0.00**
- **毛利率：0.00%**

### 7.2 原因说明
本次数据中，销售金额与“销售数量 × 商品单价”高度一致，因此反推出的毛利为零。这通常意味着：
1. 销售金额可能按标准价直接生成；
2. 数据中未包含采购成本、折扣、返利等影响利润的字段；
3. 当前数据不适合直接用于真实盈利分析。

### 7.3 管理建议
若未来需要开展更具业务价值的利润分析，建议补充以下字段：
- 采购成本
- 实际成交价

- 运输及仓储成本

只有在具备完整成本链条后，才能准确分析毛利结构和盈利能力。
